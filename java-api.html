Java API

Важно

Доступ к Java-классам ограничен "белым списком" (allowlist) из соображений безопасности. Если класс не разрешён, java_class вернёт null.

Получение Java-класса

# Получение Java-класса по полному имени
val RandomClass = java_class("java.util.Random")
val Math = java_class("java.lang.Math")
val Lifecycle = java_class("net.minecraftforge.server.ServerLifecycleHooks")

# Вызов статических методов
val maxValue = Math.max(1, 5)
chat("Максимум: " + wholeStr(maxValue))

# Получение сервера
val server = Lifecycle.getCurrentServer()

Создание экземпляра Java-класса

val RandomClass = java_class("java.util.Random")
val randomObj = java_new(RandomClass)

# Вызов Java-методов напрямую
val randomInt = randomObj.nextInt(100)
chat("Случайное число из Java: " + wholeStr(randomInt))

Работа с сущностями через .java()

# Получение Java-объекта сущности
val javaPlayer = player.java()
val javaNPC = readix_avatar.java()

# UUID
val uuid = player.java().getUUID().toString()

# Отправка пакета игроку
val SetCameraPacket = java_class("net.minecraft.network.protocol.game.ClientboundSetCameraPacket")
val packet = java_new(SetCameraPacket, camera_npc.java())
sendPacket(player, packet)

Отправка Java-пакетов

# Создание и отправка пакета
val SetCameraPacket = java_class("net.minecraft.network.protocol.game.ClientboundSetCameraPacket")

Полный пример

# Работа со случайными числами
val RandomClass = java_class("java.util.Random")
val randomObj = java_new(RandomClass)
val randomInt = randomObj.nextInt(100)
chat("Случайное число: " + wholeStr(randomInt))

# Работа с Math
val Math = java_class("java.lang.Math")
chat("Максимум из 1 и 5: " + wholeStr(Math.max(1, 5)))

# Работа с сервером
val Lifecycle = java_class("net.minecraftforge.server.ServerLifecycleHooks")
val server = Lifecycle.getCurrentServer()

# Отправка пакета смены камеры
