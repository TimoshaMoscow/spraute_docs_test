UI система

Создание UI шаблона

create ui dialog_panel {
    size = [300, 128]
    background = "#F0181828"
    can_close = true
    
    text("q_text", "Нужна твоя помощь!") {
        pos = [12, 14]
        color = "#EAEAEA"
        scale = 1.0
    }
    
    button("ans_help", "Помогу") {
        pos = [12, 48]
        size = [158, 22]
        color = "#55336688"
        hover = "#66447799"
        onClick {
            chat("Выбрана помощь!")
        }
    }
    
    entity("npc:readix_avatar") {
        pos = [188, 30]
        size = [96, 98]
        scale = 1.05
        crop = [0, 0.0, 0.0, 0.4]
        anchor_y = 2.1
        nameTag = false
        noLookAt = true
        noFollowCursor = true
        noHurtAnim = true
    }
}

Открытие и закрытие UI

# Открыть UI
uiOpen(player, dialog_panel)

# Закрыть UI
uiClose(player)

# Открыть как HUD (без паузы)
overlayOpen(player, hud_template)
overlayClose(player)

# Открыть UI, привязанный к блоку (сохраняет предметы в слотах)
openBlockUi(player, x, y, z, block_ui)

Обновление виджетов

# Обновление текста
uiUpdate(player, "my_text", "text", "Новый текст")

# Обновление цвета кнопки
uiUpdate(player, "my_btn", "color", "#FF0000")

# Обновление ширины
uiUpdate(player, "xp_fill", "w", "30%")

Анимации

# Плавное перемещение кнопки
uiAnimate(player, "my_btn", "y", 150, 1.5, "bounce_out")

# Типы easing:
# linear, ease_in, ease_out, ease_in_out, bounce_out, elastic_out

Виджеты

text

Текст с wrap, max_lines, anchor

input

Поле ввода с placeholder, password

button

Кнопка с onClick, color, hover, texture

image

Изображение с поддержкой 9-slice

entity

3D модель с crop, anchor, nameTag, noLookAt

item/block

Иконка предмета/блока

rect/panel

Прямоугольник с цветом

scroll

Область с прокруткой

clip

Обрезка дочерних виджетов

grid_bg

Фоновая сетка (h/v/hv)

slot

Слот для предметов (в контейнере)

player_inventory

Инвентарь игрока

Контейнеры (Container UI)

create ui my_workbench {
    type = "container"
    size = [176, 166]
    background = "#00000000"
    
    image("bg") {
        pos = [0, 0]
        size = [176, 166]
        texture = "minecraft:textures/gui/container/dispenser.png"
    }
    
    player_inventory {
        pos = [8, 84]
    }
    
    slot(0) { pos = [62, 20] }
    slot(1) { pos = [80, 20] }
    slot(2) { pos = [98, 20] }
    
    button("craft_btn", "Скрафтить") {
        pos = [62, 50]
        size = [54, 20]
    }
}

В контейнерных UI предметы в слотах сохраняются в инвентарь блока. Используйте openBlockUi для открытия.

Динамическое создание виджетов

create ui dynamic_ui {
    size = [400, 200]
    background = "#FF000000"
    
    var i = 0
    while (i < listSize(items)) {
        val item = list_get(items, i)
        button("btn_" + i, item["name"]) {
            pos = [10, i * 24]
            size = [180, 20]
        }
        i = i + 1
    }
}

Проценты и позиционирование

# Панель на весь экран
create ui fullscreen {
    size = ["100%", "100%"]
    pos = [0, 0]
}

# По центру
text("centered", "По центру") {
    pos = ["50%", "50%"]
    anchor = "center"
}

# Снизу по центру (оверлей)
create ui bottom_hud {
    size = [300, 80]
    pos = ["center", -40]
}
