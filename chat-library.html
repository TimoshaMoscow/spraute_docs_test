Библиотека кастомного чата

Подключение

startScript("spraute_chat")
import("spraute_chat")

startScript запускает скрипт чата в фоне (регистрирует обработчик клавиши B). import импортирует функцию npc_chat.

Настройка головы NPC

create npc my_guard {
    name = "Охранник"
    hp = 20
    pos = [0, 64, 0]
}
my_guard.head = "minecraft:textures/heads/guard.png"

Отправка сообщения

npc_chat(player, my_guard, "Проход закрыт, уходи отсюда!", "#FF5555")

Сообщение показывается над хотбаром с анимацией прозрачности и автоматически исчезает через 5 секунд. При нажатии B открывается история последних 50 сообщений.
