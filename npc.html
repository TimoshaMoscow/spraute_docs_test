Создание NPC

Базовый синтаксис

create npc readix_avatar {
    name = "ReadixGG"
    model = "geo/defolt.geo.json"
    texture = "textures/entity/defolt.png"
    pos = 8, -60, 7
    show_name = false
    speed = 0.2
    hp = 20
    collision = true
    dimension = "minecraft:overworld"
}

Свойства NPC

  • name — отображаемое имя
  • hp — здоровье и максимальное HP
  • speed — скорость движения
  • dimension — измерение для спавна ("minecraft:overworld", "minecraft:the_nether", "minecraft:the_end")
  • pos — координаты x, y, z (числа через запятую, БЕЗ скобок)
  • rotate — углы yaw, pitch (числа через запятую)
  • show_name — показывать имя (по умолчанию true)
  • collision — наличие коллизии (по умолчанию true)
  • model, texture — ресурсы модели и текстуры
  • idle_anim, walk_anim — анимации
  • drop_item — ID предмета, выпадающего при смерти
  • drop_min, drop_max — мин/макс количество дропа
  • drop_chance — шанс выпадения (0-100)

Анимации

# Проигрывание анимации один раз
npc.playOnce("hello")

# Зацикленная анимация
npc.playLoop("sad")

# Заморозка на последнем кадре
npc.playFreeze("wave")

# Остановка анимации
npc.stop("animName")

# Сброс всех оверлейных анимаций
npc.stopOverlay()

# Процедурные анимации (дыхание, дрожание)
npc.setAdditiveWeight(1)

Второй аргумент анимации: true/"additive" — накопительное смешивание, false/"replace" — замена.

Движение

# Перемещение к точке (с await блокирует до прибытия)
await npc.moveTo(5, -60, 15)

# Постоянное следование
npc.alwaysMoveTo(player)
npc.stopMove()

# Следование до расстояния
await npc.followUntil(player, 2.0)

Взгляд

# Разовый поворот головы
npc.lookAt(player)

# Постоянное слежение
npc.alwaysLookAt(player)
npc.stopLookAt()

# Установка кости головы
npc.setHeadBone("head")

Предметы в руках

# Установка предмета
npc.setItem("right", "minecraft:iron_sword")
npc.setItem("left", "minecraft:shield", "{CustomModelData:1}")

# Очистка
npc.removeItem("right")
npc.removeItem()  # обе руки

Кастомный дроп

# Добавление предмета в лут при смерти
npc.addDrop("minecraft:iron_ingot", 1, 3, 50)

# Выброс предмета из NPC в мир
npc.drop("minecraft:apple", 5, true)

Подбор предметов

# Подбирать только от указанного игрока
npc.pickupOnlyFrom(player)

# Снять фильтр
npc.pickupAny()

# Количество предметов
val iron = npc.countItem("minecraft:iron_ingot")

Удаление NPC

npc.remove()
