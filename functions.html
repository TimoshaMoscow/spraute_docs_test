Пользовательские функции и Await

Пользовательские функции

fun name(a, b) {
    return a + b
}

# Вызов
val result = name(1, 2)

Await — ожидание событий

await time(seconds)

Пауза на секунды

await interact(entity)

Ожидание взаимодействия

await keybind("key")

Ожидание клавиши (g, f, space, shift)

await death(target)

Смерть сущности

await uiClick(player)

Клик в UI (возвращает id или "__close__")

await uiInput(player, widget_id)

Ввод текста в input

await uiClose(player)

Закрытие UI

await pickup(npc, amount, item_id)

Подбор предметов NPC

await orbPickup(player, amount, texture)

Подбор кастомных орбов

await position(player, x, y, z, radius)

Вход в радиус

await chat(player, message)

Сообщение в чат

await clickBlock(player, target)

Клик по блоку

await breakBlock(player, target)

Разрушение блока

await placeBlock(player, target)

Установка блока

await task("task_id")

Завершение асинхронной задачи

Async — асинхронные задачи

# Запуск с id
async "readix_idle" {
    readix_avatar.alwaysLookAt(knight)
    while (true) {
        await time(5)
        # действие каждые 5 секунд
    }
}

# Остановка
stop_task("readix_idle")

# Проверка завершения
if (taskDone("task_id")) {
    chat("Задача выполнена!")
}

# Ожидание
await task("readix_idle")

await uiClick внутри async не поддерживается.
