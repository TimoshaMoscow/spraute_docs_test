События (on) и таймеры (every)

Синтаксис

on event(params) -> handler_id {
    # код обработчика
}

every(seconds) -> handler_id {
    # код, выполняемый каждые seconds секунд
}

# Остановка обработчика
stop(handler_id)

# Отмена события (например, отмена установки блока)
cancelEvent()

События взаимодействия

on interact(target) -> id { }

При взаимодействии с target. В теле: _event_player

on keybind("key") -> id { }

Нажатие клавиши. _event_player

on death(target) -> id { }

Смерть сущности. _event_entity, _event_killer

События предметов и орбов

on pickup(npc, item_id) -> id { }

Подбор предмета NPC. _event_npc, _event_item, _event_dropper

on inventory(player, item_id, count) -> id { }

Появление предмета у игрока. _event_player

on orbPickup(player, texture) -> id { }

Подбор кастомного орба. _event_player, _event_texture, _event_amount

События UI

on uiClick([player]) -> id { }

Клик по кнопке. _event_player, _event_widget

on uiClose([player]) -> id { }

Закрытие UI. _event_player

on uiInput([player], [widget_id]) -> id { }

Ввод в input. _event_player, _event_widget, _event_input

События позиции и блоков

on position(player, x, y, z, r) -> id { }

Вход в радиус. _event_player

on clickBlock([target]) -> id { }

Клик по блоку. _event_player, _event_x, _event_y, _event_z, _event_block, _event_action

on breakBlock([target]) -> id { }

Разрушение блока. _event_player, _event_x, _event_y, _event_z, _event_block

on placeBlock([target]) -> id { }

Установка блока. Можно отменить через cancelEvent(). _event_player, _event_x, _event_y, _event_z, _event_block

События чата

on chat([player], [message]) -> id { }

Сообщение в чат. _event_player, _event_message

Примеры

# Обработка подбора железа
on pickup(readix_avatar, "minecraft:iron_ingot") -> progress {
    val have = readix_avatar.countItem("minecraft:iron_ingot")
    chat("Собрано: " + have + " слитков")
}

# Остановка
stop(progress)

# Таймер
every(5) -> timer {
    chat("Это сообщение каждые 5 секунд")
}

# Отмена установки блока
on placeBlock("minecraft:tnt") -> cancel_tnt {
    cancelEvent()
    chat("§cНельзя ставить динамит!")
}

# Обработка клика по кнопке в магазине
on uiClick() -> shop_handler {
    val player = _event_player
    val btn = _event_widget
    
    if (btn == "buy_sword") {
        giveItem(player, "minecraft:iron_sword", 1)
    }
}
